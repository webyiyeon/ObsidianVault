
# 보건

### 건강관리실
#### 보건상담
- 보건상담내용 등록관리
- 분류: 유소견자 상담 / 보건 상담 (각 양식, 질환별 분류는 필요 없음)
- 목록에서 승인요청여부 확인
- 상세 화면
	- 부서, 이름
	- 콜레스테롤 수치 등 검진 결과 
	- 상담 체크박스 및 질문지(예: 흡연 유무, 음주 유무)
	- 보건 관리자 서명
	- 상담자 서명
	- 보건 관리자 서명 후, 상담 완료 시 상담자에게 서명 요청 메일 및 알람. 
	  상담자까지 서명 시 최종 완료.
	- 위 사항으로 모바일 화면 필요


#### 건강검진결과 등록
- PDF 파일 일괄 등록
	- 등록 시, 부서/이름/연령 으로 인원 파악
	- 양식: 동강병원, 타 병원 (총 2개) 외에는 key-in 방식
- 목록에서 해당 인원 상담 이력 관리 연계(보건상담)
- 유관찰자 대상 메일 발송
- 기준정보: 질환-상담 매칭 조건(제공 받아야 함)

#### INBODY 측정 결과 등록
- key-in으로만 가능
- 기계 X 

#### 통계관리
- 팀별 유소견자 현황 조회 ⇒ 더블 클릭 시, 상담으로 이동 가능
- 차후 상세 결정
- 건강증진프로그램 통계 조회
