
# 보건

## 건강관리실
### 보건상담
- 보건상담내용 등록관리
- 분류: 유소견자 상담 / 보건 상담 (각 양식, 질환별 분류는 필요 없음)
- 목록에서 승인요청여부 확인
- 상세 화면
	- 부서, 이름
	- 콜레스테롤 수치 등 검진 결과 
	- 상담 체크박스 및 질문지(예: 흡연 유무, 음주 유무)
	- 보건 관리자 서명
	- 상담자 서명
	- 보건 관리자 서명 후, 상담 완료 시 상담자에게 서명 요청 메일 및 알람. 
	  상담자까지 서명 시 최종 완료.
	- 위 사항으로 모바일 화면 필요

### 건강검진결과 등록
- PDF 파일 일괄 등록
	- 등록 시, 부서/이름/연령 으로 인원 파악
	- 양식: 동강병원, 타 병원 (총 2개) 외에는 key-in 방식
- 목록에서 해당 인원 상담 이력 관리 연계(보건상담)
- 유관찰자 대상 메일 발송
- 기준정보: 질환-상담 매칭 조건(제공 받아야 함)

### INBODY 측정 결과 등록
- key-in으로만 가능
- 기계 X 

### 통계관리
- 팀별 유소견자 현황 조회 ⇒ 더블 클릭 시, 상담으로 이동 가능
- 차후 상세 결정
- **건강증진프로그램** 통계 조회 

### 프로그램/질환별 관리요령 홍보
- 이메일 발송 
- SMS 발송(기존 유비스 시스템과 동일)
- 대상: 유관찰자

### 건강증진프로그램
- **건강증진프로그램** 등록
- 개개인이 직접 설문조사 참가하여 본인의 정도 파악 가능 및 보건관리자 통계 확인


## 작업환경측정
### 작업환경측정
- 연 2회 or MSDS 신규 추가 시 작업환경측정 필요
- PDF 첨부파일을 통한 결과 데이터만 관리
- 메일/문자(전사): 작업환경측정 완료되었습니다.

## 근골격계 부담작업조사
### 근골격계 부담작업조사
- 보건관리자가 연도별 데이터 생성 시, 자동으로 전 부서 항목 생성
- 각 부서는 직접 결과 작성 후 제출 (임시저장, 제출)
- 잘못 작성되었을 경우, 보건관리자와 소통 후 수정 (재작성 요청)
- 한 건이라도 체크될 경우 근골격게 부담작업에 해당, 모두 X일 때만 해당없음
- 정기조사: 3년에 1회 / 수시조사: 사고 시
- 첨부파일 유지
- 각 이미지에 대한 상세 내용 추가(클릭 시 팝업 or 툴팁 등 작성요령 보이게)
