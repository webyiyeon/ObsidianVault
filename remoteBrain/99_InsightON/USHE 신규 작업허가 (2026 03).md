
## 규칙

#### 기본 전제

- [x] 일반 작업과 안전 작업은 동시에 선택할 수 없다.
- [x] Blind 설치 해체 작업은 일반 작업에도 있고 안전 작업에도 있다.
- [x] 전기, 밀폐 공간은 안전 작업에 속한다.
- [x] 기압 Test와 탱크로리 입출하 작업은 일반 작업에 속한다.
- [x] 탱크로리 입출하 작업은 첨부를 발생시키는 조건이 아니라,
      취급 화학물질과 함께 선택되었을 때 공통 첨부 체크를 해제하는 조건이다.
- [x] 전기 작업은 다른 조건보다 우선 적용된다.

### 1. 공통 첨부 자동 체크 기준

- [ ] LOTO,Blind ,P&ID List 첨부  
      다음 항목 중 하나라도 선택되면 안전 조치사항 공통의 첨부 항목이 자동 체크되어야 한다.
  - [ ] 일반 작업의 기압 Test
  - [ ] 일반 작업의 Blind 설치 해체 작업
  - [ ] 안전 작업의 밀폐 공간
  - [ ] 안전 작업의 Blind 설치 해체 작업
  - [ ] 취급 화학물질 9개 항목 중 하나 이상
- [ ] 위 조건이 모두 해제되면 위 항목도 자동 해제되어야 한다.
- [ ] 일반위험작업에서 **유해위험물질 저장, 취급, 사용 선택**이 되어있는 경우,
      **기압 Test, Blind 설치 해제 작업** 체크 해제 시 자동으로
      안전조치사항 > 공통 > LOTO List, Blind List, P&ID 첨부가 해제되지 않음

### 2. 취급 화학물질과 탱크로리 예외 조건

- [x] 취급 화학물질 9개 중 하나라도 선택되면 공통 첨부 항목은 자동 체크된다.
- [x] 다만 취급 화학물질과 일반 작업의 탱크로리 입출하 작업이 동시에 선택된 경우에는 공통 첨부 항목이 자동 해제되어야 한다.
- [x] 즉, 취급 화학물질은 공통 첨부를 발생시키는 조건이고, 탱크로리는 단독으로는 아무 첨부도 발생시키지 않으며, 취급 화학물질과 함께 있을 때만 공통 첨부를 해제하는 조건이다.
- [x] 따라서 일반 작업에서 탱크로리만 선택된 경우에는 공통 첨부도 없고, 별도의 필수 첨부도 없다.

### 3. MSDS 첨부 기준

- [x] 안전 조치사항 공통에서 취급 화학물질 MSDS 교육 첨부를 선택한 경우 MSDS 파일은 필수로 첨부되어야 한다.

### 4. 허가부서 단계 필수 첨부 기준

- [x] 허가부서 단계에서는 **Blind 설치 해체 작업 선택 여부**를 기준으로 필수 첨부를 판단한다.
- [x] 여기서 Blind 설치 해체 작업은 일반 작업에서 선택된 경우와 안전 작업에서 선택된 경우를 모두 포함한다.
- [x] Blind 설치 해체 작업이 선택된 경우에는 다음 두 항목이 필수 첨부이다.
  - Blind List
  - P&ID
- [x] Blind 설치 해체 작업이 선택되지 않은 경우에는 다음 두 항목이 필수 첨부이다.
  - LOTO List
  - P&ID
- [x] 다만 이 허가부서 필수 첨부 기준은 기압 Test, Blind 설치 해체 작업, 밀폐 공간, 취급 화학물질 등으로 인해 첨부 필요 조건이 발생한 경우를 전제로 적용해야 한다.
- [x] 즉, 탱크로리 단독 선택처럼 애초에 첨부 필요 조건이 없는 경우에는 허가부서에서도 필수 첨부가 발생하지 않는다.
- [x] 필수 첨부를 건너뛸 경우에는 사유를 입력해야 하며, 입력된 사유는 검토의견에 자동 삽입되어야 한다.

### 5. 신청부서 단계 전기 작업 기준

- [x] 안전 작업에서 전기 동력 차단 작업을 선택한 경우에는 안전 조치사항 전기의 다음 두 항목이 자동 체크되어야 한다.
  - LOTO List 첨부
  - 전기 단선도 첨부
- [x] 위 두 항목은 신청부서 단계에서 필수 첨부이다.
- [x] 필수 첨부를 건너뛸 경우에는 사유를 입력해야 하며, 입력된 사유는 검토의견에 자동 삽입되어야 한다.
- [x] 전기 작업은 다른 조건에 의해 해제되지 않는다.

### 6. 전기 작업과 다른 조건의 관계

- [x] 전기 작업은 항상 최우선으로 적용된다.
- [x] 따라서 전기 작업이 선택된 경우에는 다른 조건과 관계없이 신청부서 단계에서 LOTO List와 전기 단선도는 반드시 유지되어야 한다.
- [x] 다만 전기 작업과 함께 밀폐 공간, Blind 설치 해체 작업, 취급 화학물질이 선택될 수 있으므로, 이 경우에는 전기 첨부와 공통 첨부 로직이 함께 적용될 수 있다.
- [x] 예를 들어 **전기**와 **밀폐 공간**이 함께 선택된 경우에는 **공통 첨부 세 항목**이 자동 체크되고,
      신청부서에서는 **LOTO List와 전기 단선도**가 필수이며,
      허가부서에서는 Blind 작업이 아니므로 **LOTO List와 P&ID**가 필수이다.
- [x] **전기**와 **Blind 설치 해체 작업**이 함께 선택된 경우에는 **공통 첨부 세 항목**이 자동 체크되고,
      신청부서에서는 **LOTO List와 전기 단선도**가 필수이며,
      허가부서에서는 **Blind List와 P&ID**가 필수이다.
- [x] 전기와 취급 화학물질이 함께 선택된 경우에도 전기 첨부는 그대로 유지되며, 공통 첨부는 취급 화학물질에 의해 체크된다.
- [x] 전기와 탱크로리는 함께 선택될 수 없다. 전기는 안전 작업이고 탱크로리는 일반 작업이기 때문이다.

### 7. 대표 경우의 수

- [ ] 일반 작업에서 기압 Test 선택 ⇒ `confirm();`
      공통 첨부는 LOTO List, Blind List, P&ID가 체크된다.  
      허가부서 필수 첨부는 LOTO List와 P&ID이다.

- [ ] 일반 작업에서 Blind 설치 해체 작업 선택
      공통 첨부는 LOTO List, Blind List, P&ID가 체크된다.  
      허가부서 필수 첨부는 Blind List와 P&ID이다.

- [ ] 일반 작업에서 탱크로리만 선택
      공통 첨부는 체크되지 않는다.  
      허가부서 필수 첨부도 없다.

- [ ] 일반 작업에서 취급 화학물질 선택
      공통 첨부는 LOTO List, Blind List, P&ID가 체크된다.  
      허가부서 필수 첨부는 LOTO List와 P&ID이다.

- [ ] 일반 작업에서 취급 화학물질과 탱크로리를 함께 선택
      공통 첨부는 해제된다.  
      허가부서 필수 첨부도 발생하지 않는다.

- [ ] 안전 작업에서 밀폐 공간 선택
      공통 첨부는 LOTO List, Blind List, P&ID가 체크된다.  
      허가부서 필수 첨부는 LOTO List와 P&ID이다.

- [ ] 안전 작업에서 Blind 설치 해체 작업 선택 ``
      공통 첨부는 LOTO List, Blind List, P&ID가 체크된다.  
      허가부서 필수 첨부는 Blind List와 P&ID이다.

- [ ] 안전 작업에서 전기 선택
      신청부서 필수 첨부는 LOTO List와 전기 단선도이다.  
      공통 첨부는 전기만으로는 체크되지 않는다.  
      허가부서 필수 첨부도 전기만으로는 발생하지 않는다.

- [ ] 안전 작업에서 전기와 밀폐 공간 선택
      공통 첨부는 LOTO List, Blind List, P&ID가 체크된다.  
      신청부서 필수 첨부는 LOTO List와 전기 단선도이다.  
      허가부서 필수 첨부는 LOTO List와 P&ID이다.

- [ ] 안전 작업에서 전기와 Blind 설치 해체 작업 선택
      공통 첨부는 LOTO List, Blind List, P&ID가 체크된다.  
      신청부서 필수 첨부는 LOTO List와 전기 단선도이다.  
      허가부서 필수 첨부는 Blind List와 P&ID이다.

- [ ] 안전 작업에서 전기와 취급 화학물질 선택
      공통 첨부는 LOTO List, Blind List, P&ID가 체크된다.  
      신청부서 필수 첨부는 LOTO List와 전기 단선도이다.  
      허가부서 필수 첨부는 LOTO List와 P&ID이다.
