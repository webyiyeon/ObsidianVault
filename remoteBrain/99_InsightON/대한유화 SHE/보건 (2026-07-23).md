## 건강관리실

### 건강검진결과등록 
- 병원 전달 데이터
	+ 각 유해인자(질환)별로 등급이 매겨져서 전달
		+ A/B: 정상
		+ C: 요관찰자 (C1, C2)
		+ D: 유소견자 (D1, D2)
	+ 분량 300~400pages
	+ 유소견자 비율: 온산공장 전체 20명 정도 / 울산공장 전체 5명 정도
+ 건강검진 유형
	+ 일반(전사)
	+ 배치전(인사이동 시)
	+ 특수 ⇒ 문자,메일로 요청
		+ 다루는 유해물질에 따라 차이나는 항목
		+ 유해물질은 부서별로 차이 존재
		+ 프로세스
			1. 부서별 유해물질에 따라 특검 대상자 선정(보건 관리자)
			2. 결과 등록(pdf)
	+ 온산공장
		+ 사원 입사 시 1회 / 2개월 이내 1회 / 6개월 이내 1회 (배치전)
		+ 특수 1회/6개월
	+ 울산공장
		+ 사원 입사 시 1회 / 6개월 이내 1회 (배치전)
		+ 특수 1회/1년
+ 뇌심혈관의 경우, 
	+ 보고서: 1회/2년
	+ 보건 상담 시 위험도 보이게 처리 필요 (컬럼)
+ 요유관찰자인 경우,
	+ 유소견자: 상담
	+ 요관찰자: 안내문
	+ 유소견자 + 요관찰자: 상담 + 안내문 
	+ 상담 스케쥴 관리
		+ 네이버 미용실 예약과 유사한 형태 
		+ 상담은 1회만 진행(10~20분)


### 민감군 자가테스트
- 보건관리자가 대상자 선택 후 문자 및 알림 발송
- 체감온도 38℃ 이상일 때 발송 
- 현재 안내 문자 발송 대상 약 70명 정도
- 민감군 대상 선정 기준
	- 울산공장
		- 혈압, 당뇨
		- 만 55세 이상
		- 신입
	- 온산공장
		- 체크리스트 전달 후 본인이 해당되는 지 확인
		- 체크리스트: 고지혈, 뇌심혈관, 온열질환, 만 65세 이상 등 
- 법적 규제: 관리자가 확인하여 관리해라. 
- 프로세스: 
	1. 체감온도 측정
	2. 38℃ 이상일 경우 민감군+각 팀장에게 문자 발송
	3. 자가테스트 작성(필수 X)
	4. 결과 작성 시 팀장에게 안내
