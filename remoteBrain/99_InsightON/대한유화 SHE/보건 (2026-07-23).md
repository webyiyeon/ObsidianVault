## 건강관리실

### 건강검진결과등록 
- 병원 전달 데이터
	+ 각 유해인자(질환)별로 등급이 매겨져서 전달
		+ A/B: 정상
		+ C: 요관찰자 (C1, C2)
		+ D: 유소견자 (D1, D2)
	+ 분량 300~400pages
	+ 유소견자 비율: 온산공장 전체 20명 정도 / 울산공장 전체 5명 정도
+ 건강검진 유형
	+ 일반(전사)
	+ 배치전(인사이동 시)
	+ 특수 ⇒ 문자,메일로 요청
		+ 다루는 유해물질에 따라 차이나는 항목
		+ 유해물질은 부서별로 차이 존재
		+ 프로세스: 부서별 유해물질에 따라 특검 대상자 선정(보건 관리자) → 결과 등록(pdf)
	+ 온산공장
		+ 사원 입사 시 1회 / 2개월 이내 1회 / 6개월 이내 1회 (배치전)
		+ 특수 1회/6개월
	+ 울산공장
		+ 사원 입사 시 1회 / 6개월 이내 1회 (배치전)
		+ 특수 1회/1년
